# Cryptocurrency Risk Analysis

This project is a Python-based tool for analyzing the risk of different cryptocurrencies, with a focus on meme coins. It uses historical price data to compute basic risk metrics and apply simple machine learning models.

## Features

- Data loading and cleaning from CSV/JSON with historical crypto prices
- Calculation of basic risk metrics (volatility, max drawdown, simple returns)
- Comparison of multiple meme coins on the same risk scale
- Basic visualization of price dynamics and risk indicators
- (Optional) Simple ML model to classify coins by risk level

## Tech Stack

- NumPy, pandas, matplotlib / seaborn
- scikit-learn (if you used ML methods)
- Jupyter Notebook (for experiments)

## Project Structure

- `data/` – sample or demo datasets (without private API keys)
- `notebooks/` – Jupyter notebooks with experiments and analysis
- `src/` – main Python scripts (data processing, modeling, visualization)
- `README.md` – project description
- `requirements.txt` – list of Python dependencies
